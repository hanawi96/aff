✅ Fixed Issues

API URL Fixed: Moved action from query parameter to POST body
Create Category: Now sends action in request body
Delete Category: Fixed API call format
Reorder Categories: Fixed API call format

🔧 What Was Fixed

❌ Before (Causing 400 Error)

// Wrong: Action in URL query parameter
const response = await fetch(`${API_URL}?action=createCategory`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(categoryData)  // Missing action in body
});

✅ After (Fixed)

// Correct: Action in POST body
const payload = {
    action: 'createCategory',
    ...categoryData
};

const response = await fetch(API_URL, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(payload)  // Action included in body
});
