🧪 Test ShopVD Extension

Trang test extension - Không cần vào pancake.vn thật

✅ Extension đã load chưa?

Nếu sidebar màu tím xuất hiện bên phải màn hình → Extension hoạt động OK!

Nếu chưa thấy → Vào chrome://extensions/ → Load extension

📖 Hướng dẫn test:

1. Bôi đen (select) text mẫu bên dưới
2. Click nút "✨ Lấy thông tin" trên sidebar
3. Kiểm tra form tự động điền
4. Thêm sản phẩm → Click "Tạo đơn"

📝 Kịch bản test

1️⃣ Format chuẩn (có label)

Test Case #1: Khách gửi format đầy đủ

Tên: Nguyễn Văn A SĐT: 0901234567 Địa chỉ: 123 Đường Nguyễn Huệ, Phường Bến Nghé, Quận 1, TP.HCM

👆 Click để bôi đen, sau đó click nút "Lấy thông tin" trên sidebar

Test Case #2: Format viết tắt

Tên: Trần Thị B SĐT: 0912345678 DC: 456 Lê Lợi, P. 4, Q. Gò Vấp, TP.HCM

👆 Test với "DC:" thay vì "Địa chỉ:"

2️⃣ Format tự do (không label)

Test Case #3: Khách gửi text tự do

Nguyễn Văn C 0923456789 789 Võ Văn Tần, Phường 6, Quận 3, TP.HCM Giao giờ hành chính nhé shop

👆 Không có label, AI tự parse

Test Case #4: Format lộn xộn

Em đặt 2 bình sữa nha Họ tên: Phạm Thị D Số điện thoại em: 0934567890 Địa chỉ giao hàng: 321 Phan Văn Trị, Phường 7, Quận Gò Vấp, TP.HCM Giao buổi chiều giúp em

👆 Lẫn lộn với text khác

3️⃣ Edge cases

Test Case #5: SĐT có +84

Tên: Lê Văn E SĐT: +84945678901 Địa chỉ: 555 Trường Chinh, Phường 14, Quận Tân Bình, TP.HCM

👆 Test parse số điện thoại +84

Test Case #6: Địa chỉ ngắn

Hoàng Thị F 0956789012 666 Hai Bà Trưng, Quận 1, TP.HCM

👆 Địa chỉ ngắn gọn

✅ Checklist test đầy đủ

  • Sidebar xuất hiện bên phải
  • Parse text format chuẩn → Điền form OK
  • Parse text tự do → Điền form OK
  • Parse SĐT +84 → Convert về 0xxx
  • Thêm/xóa sản phẩm hoạt động
  • Tính tổng tiền chính xác
  • Checkbox "Đơn ưu tiên" hoạt động
  • Checkbox "Gửi sau" → Hiện datetime picker
  • Dropdown "Nguồn khách" có 3 option
  • Input mã CTV, mã giảm giá hoạt động
  • Nhập tiền cọc → Validate OK
  • Click "Tạo đơn" → Gọi API (check Console)
  • Sau tạo đơn → Form reset về trạng thái ban đầu

🐛 Debug & Troubleshooting

1. Mở DevTools để xem log:

F12 → Console tab → Sẽ thấy:
🚀 ShopVD Order Helper loaded

2. Xem Network request:

F12 → Network tab → Click "Tạo đơn"
→ Xem request POST tới https://ctv-api.yendev96.workers.dev/api/order/create

3. Extension không hoạt động?

① Vào chrome://extensions/
② Bật "Developer mode"
③ Click "Reload" trên extension
④ Refresh trang test (Ctrl+R)
