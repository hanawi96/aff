🍞 Toast Manager Demo

Hệ thống quản lý thông báo thông minh - Giải quyết vấn đề toast chồng lên nhau

1. Toast cơ bản

2. Nhiều toast cùng lúc (tối đa 3)

Chỉ hiển thị tối đa 3 toast, toast cũ nhất sẽ tự động ẩn

3. Toast với ID - Thay thế thông minh ⭐

✨ Điểm đặc biệt: Toast "đang xử lý" sẽ biến thành toast "hoàn thành" mượt mà, không bị chồng lên nhau!

4. Thời gian tùy chỉnh

5. Xóa tất cả toast

📝 Code Examples

// Toast cơ bản
showToast('Message', 'success');
// Toast với ID (thay thế)
showToast('Đang xử lý...', 'info', 0, 'my-action');
showToast('Hoàn thành!', 'success', null, 'my-action');
// Toast tùy chỉnh thời gian
showToast('Message', 'info', 5000); // 5 giây
